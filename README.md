<h3>Notes App - UIKit</h3>
<hr>

Notes App - UIKit is a simple notes app that uses Core Data framework for saving notes. CRUD operation handling with CoreData.
<br/><br/><br/>
<h3>Tech Used</h3>
<hr>
<ul>
  <li>Designed with MVC architecture</li>
  <li>Swift 5</li>
  <li>UIKit</li>
  <li>CoreData</li>
  <li>XCode 15</li>
</ul>
